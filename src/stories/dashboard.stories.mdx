# Dashboard

Dashboards are a simple organization tool, with some styling for flexibly arranging containers in the dashboard to fit the space.

<ArgsTable story="Dashboard" />

## DBaseDashboard component

```html
<DBaseDashboard>
  <template #main-nav></template>
  <template #main-header></template>
  <template #main-section></template>
</DBaseDashboard>
```
