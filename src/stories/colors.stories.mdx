# Colors

CCV uses the Brown University theme for colors. 
CCV's main colors are yellow and green and have varitations to improve accessibility in light and dark modes.
We created CSS variables to make the use of colors simple and standard. 
Each color swatch below contains the color, the hsl (hue, saturation, lightness), and the CSS variable.
In addition to CSS variables, this theme provides helper classes for background and text color. It's important that
you only use CSS variables or helper classes and never hardcode a color code in your element in order to guarantee the colors 
will change based on the user's mode preference (light or dark mode).

### Examples:
> using CCS variables:
```scss
background-color: var(--color-primary);
```

> using helper classes:
<!-- note: sass as language because scss was treating . in a weird way. -->
```sass 
css-selector {
    @extend .has-background-primary;
}
```
